# Node js and express rest server with typescript and mySql

## Dev

1. Copy the ```.env.example``` and rename it to ```.env```
2. Set your enviroment variables in the ```.env``` file.
3. Execute 
```
npm install 
```
4. Compile the project to javascript.
```
tsc
```
5. Execute project.
```
nodemon dist/app.js
```
